# Student Finance App

Java Desktop Application untuk pencatatan keuangan mahasiswa.

## Fitur:
- Login system
- Tambah pemasukan & pengeluaran
- Grafik (JFreeChart)
- Export PDF
- Dark mode UI

## Teknologi:
- Java Swing
- JFreeChart
- OpenPDF

## Cara Run:
javac -cp "lib/*;src" src/*.java -d bin
java -cp "bin;lib/*" Main
